# OpenVLA





* Llama 2와 같은 오픈소스 LLM을 기반으로 만든 고성능 VLA입니다.
* RT-2와 유사한 구조지만 완전한 오픈소스로 공개되었으며, 양자화(Quantization) 등을 통해 일반 GPU에서도 효율적으로 학습 및 추론이 가능하도록 최적화했습니다.
* 실제 연구나 프로젝트에 바로 가져다 쓸 수 있는 가장 실용적인 최신 모델 중 하나입니다.
