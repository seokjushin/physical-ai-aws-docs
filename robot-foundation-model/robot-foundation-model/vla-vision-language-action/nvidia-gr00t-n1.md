# NVIDIA GR00T N1



* **GR00T N1 (NVIDIA)**: 이중 시스템 아키텍처 기반
  * **System 2 (Vision-Language Module)**: 시각과 언어 지시를 통해 환경 해석
  * **System 1 (Diffusion Transformer)**: 실시간 유동적인 모터 동작 생성
  * 두 모듈은 긴밀하게 결합되어 엔드-투-엔드 공동 학습
